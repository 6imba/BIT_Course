- HTTP protocol is used when comuniacting between Web_Server and Clients while Loading pages, Submit form, Ajax calls, so on. (request and response cycle)
- Communication between Web_Server and Clients via HTTP protocol is stateless.

- As it is stateless we can proxy like statefull via programming, local storage, cookies, session, son on.
